# Microsoft Power Pages (microsoft-power-pages)
Microsoft Power Pages enables building secure, low-code business websites with Web API access to Dataverse data.

## APIs

### Power Pages Web API
The Power Pages Web API provides CRUD operations on Dataverse tables from Power Pages websites.
